### **2. Short-Address 短网址转换平台** `2023.01 - 2023.03`

_[项目地址 ：https://github.com/im-lauson/Short-Address](https://github.com/im-lauson/Short-Address)_<br>

### **项目介绍：**

该短网址转换平台是基于Go语言和Redis数据库技术开发的，为了方便用户分享长网址，并改善长网址的传播率而开发。
该平台的功能是将输入的长URL通过自实现的算法转换成6位字符长度的短URL，并使用Redis数据库存储短URL和对应的长URL，
用户使用短URL访问时，转换算法将短URL映射回长URL。同时，该平台还支持设置过期时间，提升了平台的灵活性和易用性。

**技术栈：**

1. 前端：Vue + Vue-router + ElementUI
2. 后端：Go + Mux + Redis + Docker

### **成果与亮点：**

1. 高效的短网址生成算法：使用自增序列实现短网址生成，能够快速生成短网址，提高了系统的转换效率。
2. 高效的 Redis 数据库储存：使用 Redis 数据库储存短 URL 和对应的长 URL，数据库响应速度快，存取数据效率高，提升了平台的性能和稳定性。
3. 易用的功能设置：平台支持设置过期时间、自定义短 URL 后缀等功能，简化用户使用流程，提高平台的可用性和易用性。
4. 优秀的编程风格：使用 Go
   语言这一评价极高的编程语言，提高了代码的可维护性、易读性、安全性和性能。同时，代码遵循严格的编码规范，保证了代码的质量和可靠性。<!-- <div style="page-break-after:always"></div> -->

### **3. Portal-Drug-System药品管理系统** `2021.11 - 2022.02`

### **项目介绍：**

portal-drug-system是一个将进、销、存、财务以及会员一体化专业药品管理软件。操作简单、资料管理信息化，管理方便的软件。
可随时掌握数据信息的更新情况。提供快速、准确的处理通讯录删、改、存、搜、盘点等多种功能模式， 并支持多种查看方式，为了能
更好地利用现代信息技术的成果,提高医院药品管理工作的效率和水平,而对医院药品管理系统进行改进。

### **技术栈：**

1. Go + Gin + Redis + Mysql
2. 集成viper配置管理、zap日志管理、validate参数校验、jwt认证、gorm连接数据库、swagger生成文档等技术

### **成果与亮点：**

1. 集成了viper配置管理：使用viper配置管理库来管理应用程序中的所有配置，包括API密钥、数据库配置和其他运行时选项等。方便了对应用程序的配置管理和切换。
2. 使用zap日志管理：使用zap日志库记录应用程序中的关键事件和错误，以帮助开发者更好地调试和追踪问题。同时，由于zap可以提供一系列日志级别，使得我们可以更好地控制输出的日志等级，便于调试和查错。
3. 参数校验和JWT认证：使用validate库和JWT认证实现了认证和授权。validate库可以用来校验HTTP请求中的参数和HTTP请求体数据的完整性，确保系统的输入数据合法性，并且JWT认证可以验证系统中的身份和权限，保证数据安全。
4. 连接关系型数据库：使用GORM库来连接数据库。GORM是一个ORM库，使得代码更清晰简洁、更符合面向对象开发模式，同时它也可以非常方便地对数据库进行操作。能够提高代码的可维护性和开发效率。
5. 使用Swagger生成文档：使用Swagger生成应用程序的API文档，方便非常直观、便捷地查看API接口的定义以及如何被调用。同时，也方便测试人员打开api文档进行测试功能，加速开发人员的迭代过程。
